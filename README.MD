# Gestion de l'hétérogénéité des données tabulaires pour la génération de Knowledge Graphs

Master's Thesis (PFE)

This repository contains the complete source code, datasets, LaTeX manuscript, experiments, and evaluation scripts for a modular pipeline that transforms heterogeneous tabular data into RDF Knowledge Graphs using Semantic Web technologies and Large Language Models (LLMs).

## Technologies
- Python
- Sentence-BERT
- LLaMA 3.1 (Groq)
- LangChain
- YARRRML
- Morph-KGC
- RDF / OWL / SHACL
- Splink
- Docker Compose

## Pipeline
1. Multi-source data ingestion
2. Automatic data profiling
3. Hybrid schema matching (SBERT + LLM)
4. Automatic YARRRML mapping generation
5. RDF Knowledge Graph construction
6. Entity Resolution
7. SHACL validation
8. Experimental evaluation
